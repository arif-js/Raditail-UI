## Getting Started

```bash
# Install the package
npm install raditail

# Or with pnpm
pnpm add raditail

# Or with yarn
yarn add raditail
```

## Usage

```tsx
import { Button } from 'raditail'

function App() {
  return (
    <Button variant="solid" size="md">
      Click me
    </Button>
  )
}
```
