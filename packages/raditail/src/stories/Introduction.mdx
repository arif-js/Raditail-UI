# Raditail UI

A beautiful, accessible React component library built on Radix UI and Tailwind CSS.

## Features

- 🎨 **Beautiful Design** - Modern, clean components with Tailwind CSS
- ♿ **Accessible** - Built on Radix UI primitives for excellent a11y
- 🎭 **Themeable** - Easy customization with CSS variables
- 📦 **Tree-shakeable** - Import only what you need
- 🔧 **TypeScript** - Full type safety out of the box
- 🎯 **Composable** - Flexible component APIs

## Components

Explore our comprehensive collection of components in the sidebar:

- **Button** - Versatile button with multiple variants
- **Dialog** - Modal dialogs and alert dialogs
- **Select** - Accessible select dropdown
- **Tooltip** - Informative tooltips
- **Tabs** - Tab navigation components
- And many more...

## Getting Started

```bash
# Install the package
npm install raditail

# Or with pnpm
pnpm add raditail

# Or with yarn
yarn add raditail
```

## Usage

```tsx
import { Button } from 'raditail'

function App() {
  return (
    <Button variant="solid" size="md">
      Click me
    </Button>
  )
}
```
