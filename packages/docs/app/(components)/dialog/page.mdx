# Dialog

`Dialog` wraps `@radix-ui/react-dialog` with animated overlay and responsive sizing options.

      <DialogTitle>Invite a teammate</DialogTitle>

        Send a link to collaborate in real time.
      </DialogDescription>
    </DialogHeader>
    <div className="grid gap-3">
      <label className="text-sm font-medium">
        Email address

    </div>
  </DialogContent>
</Dialog>
